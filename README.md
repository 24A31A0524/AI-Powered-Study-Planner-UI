# AI-Powered-Study-Planner-UI


📌 Description

This project is a user interface for an AI-based study planner that helps students organize their study schedule efficiently.

🚀 Features

- Task scheduling
- Clean and simple UI
- Easy navigation

🛠️ Technologies Used

- HTML
- CSS
- JavaScript



📌 How to Run

1. Download the project
2. Open index.html in browser
